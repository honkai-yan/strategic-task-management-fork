## 系统概述

### 角色定位
战略发展部是系统的最高管理者，负责：
- 制定和下发战略任务与指标
- 监控全校各部门的指标完成情况
- 审批职能部门提交的进度报告
- 查看全局数据统计和分析报告

### 权限范围
✅ 查看所有部门的数据  
✅ 创建和分配战略任务  
✅ 审批职能部门的报告  
✅ 导出全校数据报表  
✅ 切换视角查看不同部门的数据  
✅ 管理系统配置

## 登录系统

### 访问地址
```
https://blackevil.cn
```

### 登录步骤

1. **打开浏览器**
   - 推荐使用 Chrome、Edge 或 Firefox 浏览器
   - 输入系统地址并访问

2. **输入登录信息**
   - 用户名：`admin`（或您的专属账号）
   - 密码：由系统管理员提供
   
3. **点击"登录"按钮**
   - 系统会验证您的身份
   - 登录成功后自动跳转到数据看板

4. **登录状态确认**
   - 右上角显示您的部门名称："战略发展部"
   - 右上角显示您的姓名："系统管理员"
   - 可以看到消息通知图标（小铃铛）

### 登录问题处理

**问题1：忘记密码**
- 联系系统管理员重置密码
- 或使用"忘记密码"功能（如已配置）

**问题2：账号被锁定**
- 多次输入错误密码会导致账号锁定
- 等待15分钟后自动解锁
- 或联系系统管理员手动解锁

**问题3：无法访问系统**
- 检查网络连接
- 确认系统地址是否正确
- 联系技术支持

## 数据看板

数据看板是系统的首页，展示全校战略指标执行情况的总览。

### 页面布局

#### 顶部导航栏
- **左侧**：系统Logo和名称
- **中间**：
  - 年份选择器：选择查看的年度（默认2026年度）
  - 部门切换器：切换查看不同部门的视角
- **右侧**：
  - 消息通知：显示未读消息数量
  - 用户信息：显示当前登录用户
  - 退出登录

#### 主要功能区

**1. AI智能简报**
- 位置：页面顶部
- 显示内容：全校战略执行总分、严重预警任务数、完成率、健康度、响应率
- 功能：点击"立即下钻诊断"可查看详细分析

**2. 指标完成情况分布**
- 位置：页面中部左侧
- 显示内容：各职能部门的指标完成情况柱状图
- 颜色分类：绿色（超前完成）、蓝色（正常）、黄色（预警）、红色（延期）

**3. 学院指标完成情况**
- 位置：页面中部右侧
- 显示内容：各二级学院的指标完成情况
- 图表类型：堆叠柱状图

**4. 分院排名**
- 位置：页面下部
- 显示内容：各二级学院的综合得分排名
- 图表类型：横向条形图

**5. 得分构成**
- 位置：页面下部左侧
- 显示内容：全校战略执行总分的构成
- 图表类型：环形图

**6. 预警分布**
- 位置：页面下部中间
- 显示内容：预警任务的分布情况
- 图表类型：半环形仪表盘

**7. 完成情况**
- 位置：页面下部右侧
- 显示内容：指标完成情况统计
- 图表类型：环形进度图

### 数据看板操作指南

#### 1. 年份切换
- 位置：页面顶部中间
- 操作：点击年份选择器（默认显示"2026年度"）
- 功能：切换查看不同年度的数据

#### 2. 部门视角切换
- 位置：页面顶部中间（年份选择器右侧）
- 默认显示：战略发展部
- 操作：点击下拉菜单，选择要查看的部门
- 功能：切换到不同部门的视角查看数据

#### 3. 图表交互操作
- 鼠标悬停：显示详细统计数据
- 点击柱状图：下钻查看详情
- 月份筛选：使用图表上方的月份选择器

#### 4. 导出报表
- 位置：页面右上角
- 操作：点击"导出报表"按钮
- 格式：Excel格式

## 战略任务管理

战略任务管理是战略发展部的核心功能，用于创建、分配和监控战略任务。

### 访问方式
1. 点击左侧导航栏的"战略任务"菜单
2. 或在数据看板点击相关任务卡片

### 页面说明

页面顶部有两个标签：
- **数据看板**：显示全局数据统计和可视化图表
- **战略任务管理**：显示战略任务列表和管理功能

## 消息中心

消息中心用于接收和管理系统通知、预警信息和审批提醒。

### 访问方式
点击页面右上角的小铃铛图标（消息通知图标）

### 消息类型

1. **预警通知** 🔴 - 指标延期预警、里程碑逾期提醒
2. **审批通知** 📋 - 职能部门提交的进度报告待审批
3. **系统通知** 📢 - 系统维护通知、功能更新通知
4. **任务消息** 📝 - 任务分配通知、任务完成通知

---

## 个人中心

个人中心用于管理个人信息、修改密码和设置通知偏好。

### 访问方式
点击页面右上角的用户名下拉菜单，选择"个人中心"

### 功能
1. **基本信息** - 查看个人信息
2. **修改密码** - 修改登录密码
3. **通知设置** - 配置通知偏好
4. **系统偏好** - 语言、时区等设置

## 常见问题

### Q1: 为什么我看不到某些部门的数据？
**A**: 请检查部门切换器，确保选择了正确的部门。

### Q2: 图表数据不更新怎么办？
**A**: 检查网络连接，刷新页面（F5），如果问题持续，联系技术支持。

### Q3: 如何导出数据报表？
**A**: 点击页面右上角的"导出报表"按钮。

### Q4: 忘记密码怎么办？
**A**: 联系系统管理员重置密码。
